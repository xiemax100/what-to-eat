# 今天吃什么 — 需求文档

## 产品定位

帮选择困难症患者快速决定"今天吃什么"。一键随机推荐菜品，展示附近餐厅，收藏爱吃的。

## 核心功能（MVP）

1. **随机推荐**：点击大按钮，随机出一道菜（含图片、名称、简介）
2. **菜品分类**：中餐/日料/韩餐/西餐/快餐/火锅/烧烤
3. **收藏**：喜欢的菜收藏起来，不喜欢的不再出现
4. **附近餐厅**：基于当前位置，搜索附近有这道菜的餐厅（后续版本）

## 目标用户

- 每天纠结"吃什么"的上班族
- 想尝试新菜品但不知道选什么的人

## 平台

- iOS（SwiftUI）
- Android（Jetpack Compose）

## 数据源

菜品数据内置（本地 JSON），后续版本接入 API。
